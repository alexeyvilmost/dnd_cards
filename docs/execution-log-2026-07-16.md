# Журнал автономного исполнения — старт 2026-07-16

Разбивка `docs/improvement-plan-2026-07-15.md` на сессии, **критичное первым**. Исполняется
непрерывно, задача за задачей: реализация → гейт (проверенный на откате) → `vite build` →
коммит → push → отметка `ЗАКРЫТО <sha>` в KB §7.

**Дисциплина каждой задачи:**
1. Data-задача: скрипт в `scripts/content/`, dry-run → `--apply` → повторный `--apply` (идемпотентность) → `export-prod.mjs` → снапшот в коммит.
2. Гейт обязателен и **проверен на откате** (вернуть багованные данные → тест краснеет). Ложно-зелёный гейт хуже отсутствующего.
3. `cd frontend && npx vitest run` (весь набор) + `npx vite build` зелёные ДО коммита.
4. Не трогать: авторизацию (парадигма №7), визуально отображаемый код (№6), §8.5 KB (мины).

**Легенда:** ⬜ не начато · 🔄 в работе · ✅ закрыто (sha) · ⏸ отложено (причина).
