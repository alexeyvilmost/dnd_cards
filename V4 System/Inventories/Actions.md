Инвентарь действий перечисляет все доступные персонажу в данный момент действия за исключением заклинаний.

Действие представляет собой событие, требующее платы в ресурсах и дающее описываемый на нём эффект.

#### Безусловные действия

У каждого персонажа в инвентаре действий есть безусловные действия:

- Рывок (Удваивание скорости перемещения до конца хода)
- Отступление (Отход от врага не провоцирует атак)
- Засада (Скрытие от врага)
- Оттолкнуть (Отпихнуть врага от себя)
- Своё действие (Игрок описывает действие мастеру и результат применяется им)
- Безоружная атака
- Атака ближнего боя с оружием
- Дальнобойная атака оружием
- Метнуть
- Помощь
- Прыжок

Все эти действия доступны в каждом инвентаре действий. Он выглядит следующим образом:

``` showLineNumbers
"actions": [
	"action_dash",
	"action_sneak",
	"action_disengage",
	"action_help",
	"action_throw",
	"action_shove",
	"action_jump",
	"action_unarmed_strike",
	"action_melee_attack",
	"action_ranged_attack",
	"action_improvise"
]
```

Каждое действие можно посмотреть подробнее по его названию:

