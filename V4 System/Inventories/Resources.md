В этом инвентаре у персонажа хранятся ресурсы, которые он может тратить на действия.

#### Безусловные ресурсы

У каждого персонажа в это инвентаре есть пять безусловных ресурсов:

- Основное действие (1 раз/ход)
- Бонусное действие (1 раз/ход)
- Ответное действие (1 раз/ход)
- Свободное действие (1 раз/ход)
- Скорость (N футов/ход, зависит от расы)

Вот как это выглядит в инвентаре:

``` showLineNumbers
"resources": [
	"main_action",
	"bonus_action",
	"reaction",
	"free_action",
	"speed_30"
]
```

У каждого ресурса по названию можно получить подробности:

```ini showLineNumbers
GET /api/v4/resources?name="main_action"

{
	"name": "main_action",
	"russian_name": "Основное действие",
	"description": "Вы можете использовать это действие раз в ход.",
	"cooldown": "per_turn",
	"image": "main_action.png",
	"max": 1
}
```

Также сюда добавляются ресурсы от классов и предметов, вот лишь некоторые из них

- Заряды ярости
- Ячейки заклинаний
- Бардовское вдохновение
- Кости превосходства
- Скрытая атака
