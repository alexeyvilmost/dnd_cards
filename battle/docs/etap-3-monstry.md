## Что сделано

### Бэкенд
- Добавлены:
  - `backend/monsters_repo.py` — хранилище `battle_monsters`,

## Как проверить

```bash
cd battle
python3 scripts/test_battle_stack.py --no-frontend
```

Должны быть `PASS` в блоке **Monsters / PvE / XP**.

### Вручную в UI
1. `/bestiary` → «+ Создать монстра» (или редактируйте существующего).
2. `/characters` → убедитесь, что есть хотя бы один персонаж.
3. `/` (Бой) → в секции **PvE** выберите персонажа и монстра → «Старт PvE».
4. Проведите бой; ходы монстров выполняются авто-режимом.
5. После победы проверьте лист персонажа — XP должен увеличиться.

## Что важно
- PvE теперь работает на тех же боевых механиках, что и PvP.
- XP за монстров берётся из CR (или из поля `xp`, если задано явно).
